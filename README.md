## Sobre 
O objetivo principal do projeto é representar uma cadeia alimentar utilizando a estrutura de dados Grafo, além disso utilizamos o conceito de BFS para demonstrar o fluxo energético, iniciando por um nó específico(nó do produto) e terminando no animal que se deseja saber o nível trófico e utilizamos também a técinca para achar o menor caminho com o objetivo de obter o nível trófico do animal selecionado.
